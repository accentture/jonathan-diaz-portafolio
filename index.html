Algunas cosas que he construido

Red social para las Elecciones Presidenciales 2021

Una red social para expresar explicitamente porque un favorito y que espera de su gobierno, además de poder realizar un voto por tu candidato favorito.
En este proyecto tiene aplicado conceptos como autenticación, autorización y almacenamiento en la base de datos mediante comunicación HTTP entre el cliente y el servidor.
  • Angular
  • Django
  • MySQL

Simulación de multiples votos electorales

Una aplicación para hacer votaciones para las zonas y la cantidad de veces que uno desee, teniendo como resultado final las estadísticas de votación. En esta aplicación se puede retornar a la zona que votaste y continuar añadiendo votos. Esto fue posible al uso de tablas hash para cada zona.
  • HTML
  • CSS
  • Angular

Galeria de fotos usando la API de la NASA

Una web para visualizar fotos de la NASA para los años 2019 y 2020.
Este proyecto tiene implementada la arquitectura de Zakas/Osmani que consta de un núcleo, módulos, una libreria para el manejo del DOM y un sandbox para la comunicación entre componentes.
  • HTML
  • CSS
  • JavaScript

Vocabulario efectivo

Una aplicación para adquirir vocabulario rápidamente. Puedes crear tus tablas de vocabulario para cualquier documento pdf que quieras leer ya mismo, donde puedes realizar todas las operaciones de un CRUD. También puedes ver las tablas de otros usuarios y aprender de ellos.
  • HTML
  • CSS
  • Django
